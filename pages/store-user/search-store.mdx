# Search Store

Fetch all data store by query name for customer

### Endpoint

```md filename="Base URL"
http://localhost:8080
```

```md filename="Method and Endpoint"
GET /api/sellers/current/products/search
```

```md filename="Headers"
Authorization: Bearer access_token
```
