# Get Seller by Email
Fetch Seller data by Seller email

### Endpoint

```md filename="Base URL"
http://localhost:8080
```

```md filename="Method and Endpoint"
GET /api/sellers/current
```

```md filename="Header"
Authorization: Bearer access_token
```
