# Update Seller Store

Update Seller Store data

### Endpoint

```md filename="Base URL"
http://localhost:8080
```

```md filename="Method and Endpoint"
PUT /api/sellers/current/stores
```

```md filename="Headers"
Authorization: Bearer access_token
```

### Request Body
```json filename="Request Body"
{
	"Name": "store-name",
	"Description": "store-desc",
	"Logo": "logo.png",
	"Banner": "banner.png"
}
```

### Responses
You will get responses like this:

#### 200 (success)
```json filename="Response Body"
{
    "message": "Update store data successfully",
}
```

#### 404 (not found)
```json filename="Response Body"
{
    "error": "store not found"
}
```

#### 401 (unauthorized)
```json filename="Response Body"
{
    "error": "Invalid authorization header"
}
```

#### 401 (unauthorized - expire token)
```json filename="Response Body"
{
    "error": "token is expired: token is expired by 2m19s"
}
```

#### 500 (server error - failed)
```json filename="Response Body"
{
    "error": "no store was updated"
}
```

#### 500 (server error)
```json filename="Response Body"
{
    "error": "Internal server error"
}
```
