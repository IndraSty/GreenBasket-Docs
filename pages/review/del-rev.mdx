# Delete User Review

Delete user review

### Endpoint

```md filename="Base URL"
http://localhost:8080
```

```md filename="Method and Endpoint"
DELETE /api/sellers/current/review/{review_id}
```

```md filename="Headers"
Authorization: Bearer access_token
```

### Route Param

`review_id`: string (required)

### Responses
You will get responses like this:

#### 200 (success)
```json filename="Response Body"
{
    "message": "Delete Review Successfully",
}
```

#### 401 (unauthorized)
```json filename="Response Body"
{
    "error": "Invalid authorization header"
}
```

#### 401 (unauthorized - expire token)
```json filename="Response Body"
{
    "error": "token is expired: token is expired by 2m19s"
}
```

#### 404 (not found)
```json filename="Response Body"
{
    "error": "review not found with that id"
}
```

#### 500 (server error)
```json filename="Response Body"
{
    "error": "Internal server error"
}
```

#### 500 (server error - failed)
If query to `database` get troble while get the `review` data
```json filename="Response Body"
{
    "error": "failed to get review by email and id"
}
```

#### 500 (server error - failed)
If query to `database` get troble while update the `review` data
```json filename="Response Body"
{
    "error": "failed to delete the revieww"
}
```
