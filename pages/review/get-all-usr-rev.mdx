# Get All User Review by Email

Fetch all user review by user email

### Endpoint

```md filename="Base URL"
http://localhost:8080
```

```md filename="Method and Endpoint"
GET /api/sellers/current/reviews
```

```md filename="Headers"
Authorization: Bearer access_token
```
