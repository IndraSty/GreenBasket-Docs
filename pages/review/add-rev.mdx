# Add Review

Adding product review

### Endpoint

```md filename="Base URL"
http://localhost:8080
```

```md filename="Method and Endpoint"
POST /api/sellers/current/review/{order_id}
```

```md filename="Headers"
Authorization: Bearer access_token
```

### Route Param

`order_id`: string (required)

### Query Param

`product_id`: string (required)

### Request Body
```json filename="Request Body"
{
	"rating": "4.5",
	"review": "Example Review"
}
```

### Responses
You will get responses like this:

#### 200 (success)
```json filename="Response Body"
{
    "message": "Add Review Successf",
}
```

#### 400 (bad request)
```json filename="Response Body"
{
    "error": "Invalid request body",
}
```

#### 401 (unauthorized)
```json filename="Response Body"
{
    "error": "Invalid authorization header"
}
```

#### 401 (unauthorized - expire token)
```json filename="Response Body"
{
    "error": "token is expired: token is expired by 2m19s"
}
```

#### 500 (server error)
```json filename="Response Body"
{
    "error": "Internal server error"
}
```

#### 500 (server error - failed)
If query to `database` get troble while get the `order` data
```json filename="Response Body"
{
    "error": "Failed to get order by email and id"
}
```

#### 500 (server error - failed)
If query to `database` get troble while insert the `review` data
```json filename="Response Body"
{
    "error": "Failed to insert review"
}
```
