# Get All Product in Cart

Fetch all product item in user cart

### Endpoint

```md filename="Base URL"
http://localhost:8080
```

```md filename="Method and Endpoint"
GET /api/users/current/cart-items
```

```md filename="Headers"
Authorization: Bearer access_token
```
