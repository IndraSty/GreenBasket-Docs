# Get User Cart Data

Fetch user cart data

### Endpoint

```md filename="Base URL"
http://localhost:8080
```

```md filename="Method and Endpoint"
GET /api/users/current/cart
```

```md filename="Headers"
Authorization: Bearer access_token
```
