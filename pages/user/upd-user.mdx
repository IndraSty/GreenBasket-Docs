# Update User Data

Updating data user

### Endpoint

```md filename="Base URL"
http://localhost:8080
```

```md filename="Method and Endpoint"
PATCH /api/users/current
```

```md filename="Headers"
Authorization: Bearer access_token
```

### Request Body
```json filename="Request Body"
{
  "first_name": "updt-name",
  "last_name": "updt-name",
  "image_url": "https://example.com/images/example.jpg",
}
```

### Responses
You will get responses like this:

#### 200 (success)
```json filename="Response Body"
{
    "message": "Update user Successfully",
}
```

#### 400 (bad request)
```json filename="Response Body"
{
    "error": "Invalid request body"
}
```

#### 401 (unauthorized)
```json filename="Response Body"
{
    "error": "Invalid authorization header"
}
```

#### 401 (unauthorized - expire token)
```json filename="Response Body"
{
    "error": "token is expired: token is expired by 2m19s"
}
```

#### 404 (not found)
```json filename="Response Body"
{
    "error": "user not found"
}

#### 500 (server error)
```json filename="Response Body"
{
    "error": "Internal server error"
}
```

#### 500 (server error)
If something went wrong while update `user` data in `database`

```json filename="Response Body"
{
  "error": "Failed to update user"
}
```
