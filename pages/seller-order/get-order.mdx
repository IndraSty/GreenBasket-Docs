# Get Seller Order

Fetch Seller Order data

### Endpoint

```md filename="Base URL"
http://localhost:8080
```

```md filename="Method and Endpoint"
GET /api/sellers/current/orders/{order_id}
```

```md filename="Headers"
Authorization: Bearer access_token
```

### Route Param

`order_id`: string (required)
