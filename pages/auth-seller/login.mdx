# Login Seller

Authenticate Seller by Email and Password

### Endpoint

```md filename="Base URL"
http://localhost:8080
```

```md filename="Method and Endpoint"
POST /api/sellers/login
```

### Request Body
```json filename="Request Body"
{
    "email": "example@gmail.com",
    "password": "YourPassword_123",
}
```

### Responses
You will get responses like this:

#### 200 (success)
```json filename="Response Body"
{
    "access_token": "unique-string-for-access-token"
}
```

#### 400 (bad request)
```json filename="Response Body"
{
    "error": "email or password incorrect"
}
```

#### 400 (bad request - email not exist)
```json filename="Response Body"
{
    "error": "email doesn't exist"
}
```

#### 500 (server error)
```json filename="Response Body"
{
    "error": "Internal server error"
}
```
