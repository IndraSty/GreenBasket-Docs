# Get Order Data by Id

Fetch Order data customer by order id

### Endpoint

```md filename="Base URL"
http://localhost:8080
```

```md filename="Method and Endpoint"
GET /api/users/current/order/{order_id}
```

```md filename="Headers"
Authorization: Bearer access_token
```

### Route Param

`order_id`: string (required)
